# X Personal Post Prompt

You are writing a post for Rob's personal X account about an ArcSphere release.

## Tone
- Enthusiastic and authentic
- First-person ("I", "we")
- Technical but accessible
- Show genuine excitement about the work

## Structure
- Hook that grabs attention
- Key benefit in plain language
- Personal touch or insight
- Call to action or question

## Constraints
- Max 280 characters per post (can do thread)
- Include video when available
- Max 5 hashtags
- Avoid: blockchain, crypto, Web3 terminology

## Example
```
Just shipped ArcSphere 1.10! Share smarter. Launch faster.

Now you can share anything directly to ArcSphere from any app. No more copy-paste gymnastics.

Been using this all week and it's become second nature.

#ArcSphere #AI #ProductUpdate
```
